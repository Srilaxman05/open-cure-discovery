## üìò How to Use the Application

Once you have installed the application, follow these basic steps to start using it:

1. **Upload Your Data:**  
   Prepare your molecular data files. These files should be in a supported format (like .sdf or .pdb). Drag and drop or use the upload button to bring them into the application.

2. **Set Parameters:**  
   Adjust the screening settings. You can specify parameters such as target proteins and desired chemical properties.

3. **Run the Screening:**  
   Start the screening process by clicking the corresponding button. The application will process your data and highlight potential candidates based on your criteria.

4. **Review Results:**  
   View the results in the application interface. You will see a list of compounds ranked by their predicted effectiveness.

5. **Export Your Findings:**  
   If you want to save your results, use the export options to create a report or share findings with colleagues.
